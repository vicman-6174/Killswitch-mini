# Killswitch-mini

Repositorio para el robot de mini sumo autónomo Killswitch, diseñado por Víctor Manuel Zamudio Zazueta, Adolfo Javier Barrientos López y Marian Zamarripa Espinoza, con objetivo de competir en el torneo de All Japan 2026

# MiniSumo - All Japan Robot Sumo Tournament 2026

Robot autónomo diseñado para competir en el **All Japan Robot Sumo Tournament 2026**.

## Objetivo

Diseñar un minisumo competitivo priorizando:

- Alta aceleración
- Baja masa rotacional
- Alta tracción
- Estrategias configurables
- Electrónica propia (Xmotion V3)

## Especificaciones

| Componente | Valor |
|------------|-------|
| PCB | Xmotion V3 |
| Motores | N20 400 RPM |
| Sensores de línea | 2 QTR |
| Sensores de oponente | 5 IR |
| Alimentación | LiPo 3S |
| Arma | Navaja frontal |
| Chasis | Placa de latón + PLA (impresión 3D) |

## Estrategias

- Estrategia 1 — Búsqueda por sensores
- Estrategia 2 — Ataque frontal
- Estrategia 3 — Arco de reconocimiento
